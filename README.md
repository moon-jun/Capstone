# 개요
캡스톤설계 작품으로 취업에 도움이되는 매직미러를 만들었습니다.
기본적인 날씨,뉴스,시간,덕담등을 제공하며 음성제어가 가능합니다
또한 네트워크가 연결되어 있으면 유튜브 영상을 볼 수 있는 기능을 제공합니다

# 사용한 기술스택
javascript , node.js
